# underground
#Description
Our app will use google maps and user generated data to generate suggestions for how the user should spend his or her time. It will feature databases of places as well as the ability to find events/restaurants/activities/etc which can be specifically searched for or randomly given to the user. underground will also feature functions which can either create an itinerary for the user, or recommend events/restaurants/activities/etc based on the userâ€™s current itinerary.
